# Usage(java)
```
new ViewManager
.HighlightEffectBuilder(this)
.focusedOn(this.highlightCircleButton)
.highlightShape(ViewManager.SHAPE_CIRCLE)
.backgroundColor(R.color.color_00FFFFFF)
.build()
.render();
```
